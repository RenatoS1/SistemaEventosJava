│                                    PADRÕES ARQUITETURAIS                                                    │
└─────────────────────────────────────────────────────────────────────────────────────────────────────────────┘

• MVC (Model-View-Controller)
  - Model: User, Event, EventCategory
  - View: Console (via Scanner)
  - Controller: EventController

• DAO (Data Access Object)
  - UserDAO: Persistência de usuários
  - EventDAO: Persistência de eventos

• Service Layer
  - UserService: Lógica de negócio para usuários
  - EventService: Lógica de negócio para eventos
